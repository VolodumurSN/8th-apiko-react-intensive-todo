## Installing

* Clone this project (git clone ...)
* Install npm dependencies (npm install)

<br>

### Run with webpack dev server on localhost:4000
`npm run start`

### Run development mode
`npm run dev`

### Run production mode
`npm run build`

### Run watch dev mode 
"watch" all files changes <br>
`npm run watch`
